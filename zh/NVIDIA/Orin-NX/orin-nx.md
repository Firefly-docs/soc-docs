# Orin NX

## 主要特性

- NVIDIA Jetson Orin NX 小尺寸高性能边缘 AI 模组，面向机器人、无人机、机器视觉、智能零售和工业边缘计算
- Arm Cortex-A78AE CPU 搭配 NVIDIA Ampere GPU 和 Tensor Core，提供高能效 AI 推理能力
- 集成 NVDLA 深度学习加速器，适合多路视频分析和多模型并发推理
- 支持 LPDDR5 内存、高速摄像头、PCIe、USB 和显示扩展
- 兼容 Jetson 软件生态，支持 JetPack、CUDA、TensorRT、DeepStream 和 Isaac ROS

## 详细参数

| Specification | Details |
| :--- | :--- |
| **CPU** | • Up to 8-core Arm Cortex-A78AE CPU |

| **Interface** | • Supports PCIe, USB, display and peripheral expansion |
| **Software** | • Supports NVIDIA JetPack, CUDA, TensorRT, DeepStream and Isaac ROS |
