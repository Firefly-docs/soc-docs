# RK182x

## 主要特性

- 面向人工智能应用的高性能协处理器芯片
- 内置多核RISC-V CPU与2.5/ 5GB DRAM
- 集成多核高算力NPU，支持3B/ 7B大语言模型及传统CNN推理的本地化部署，具备多模态处理能力
- 支持通过PCIe 2.0、USB 3.0接口与主处理器高速互联

## 详细参数 

| Specification | Details |
| :--- | :--- |
| **CPU** | • Multi-core RISC-V |
| **NPU** | • Support 3B/7B models• Support for multiple computational precisions ranging from INT4 to FP16 (INT4/INT8/INT16/FP8/FP16/BF16) |
| **Memory** | • 2.5GB/5GB DRAM• eMMC 4.51 |
| **Media** | • RGA• 4K JPEG Decoder |
| **Connectivity** | • USB3.0/2x PCIe2.1/RGMII |
