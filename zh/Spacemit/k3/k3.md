# K3

## 主要特性

- 进迭时空新一代 64 位 RISC-V 处理器平台，面向高性能计算、边缘计算和智能终端等场景
- 基于 SpacemiT 自研 X100 RISC-V CPU 内核，兼容 RVA23 相关规范
- 支持多核集群扩展，满足从端侧设备到高性能平台的不同算力需求
- 面向 Linux / Bianbu 等 RISC-V 软件生态，便于系统和应用迁移
- 具体 GPU、NPU、多媒体和高速接口能力以最终芯片规格书及板级设计为准
