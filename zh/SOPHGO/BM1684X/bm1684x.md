# BM1684X

## 主要特性

- SOPHGO 高性能 AI 推理处理器，面向边缘 AI 服务器、视频分析、智能安防和工业视觉
- 相比 BM1684 提供更高 TPU 推理性能，适合更大模型和更高并发的视觉任务
- 支持 INT8、FP16、FP32 等多精度推理，兼顾性能、精度和部署灵活性
- 支持多路视频编解码、图像预处理和 AI 推理流水线
- 适配 SOPHON SDK 和 BModel 部署流程，便于从主流训练框架迁移到端侧推理

## 详细参数

| Specification | Details |
| :--- | :--- |
| **AI** | • Integrated high-performance SOPHGO TPU• Supports INT8 / FP16 / FP32 inference workloads• Suitable for high-concurrency CV inference and edge AI services |
| **Video** | • Supports multi-channel video decode and encode applications• Channel count depends on card, module and software configuration |
| **Image Processing** | • Supports image preprocessing, scaling, color conversion and AI pipeline acceleration |
| **Host Interface** | • Supports PCIe accelerator card, module and embedded product designs |
| **Memory** | • Supports high-bandwidth memory configuration for AI inference• Capacity depends on product design |
| **Software** | • Supports SOPHON SDK• Supports model conversion, BModel compilation, runtime inference and multimedia pipeline development |
