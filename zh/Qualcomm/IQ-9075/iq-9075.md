# IQ-9075

## 主要特性

- Qualcomm Dragonwing 智能物联网处理器平台，面向工业控制、机器人、边缘 AI、机器视觉和多媒体终端
- 64 位多核 CPU 架构，兼顾高性能计算、实时任务和低功耗场景
- 集成 Qualcomm AI Engine / Hexagon 处理器，适用于端侧视觉、语音、传感器融合和生成式 AI 推理
- Qualcomm Adreno GPU 支持图形渲染、通用计算和多媒体加速
- Qualcomm Spectra ISP 支持多摄像头输入、HDR 和高质量图像处理
- 面向工业产品设计，支持高速存储、显示、摄像头和丰富外设扩展
