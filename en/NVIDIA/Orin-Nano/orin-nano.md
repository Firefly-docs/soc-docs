# Orin Nano

## Key features

- NVIDIA Jetson Orin Nano entry-level edge AI module for robotics, smart cameras, visual inspection, education and lightweight industrial devices
- Arm Cortex-A78AE CPU with NVIDIA Ampere GPU and Tensor Cores for AI inference in a compact form factor
- Supports multi-camera input, video processing, LPDDR5 memory and high-speed peripheral expansion
- Low-power design for fanless or compact edge devices; final performance depends on module SKU and power mode
- Supports JetPack, CUDA, TensorRT, DeepStream and common AI deployment workflows
