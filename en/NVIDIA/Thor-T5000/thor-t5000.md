# Thor-T5000

## Key features

- NVIDIA Jetson Thor T5000 next-generation edge computing platform for robotics and physical AI, targeting humanoid robots, AMRs, industrial automation and multimodal AI
- NVIDIA Blackwell GPU architecture with Tensor Cores for large models, multimodal inference and real-time perception-planning workloads
- Arm Neoverse-V3AE CPU cluster for high-performance general-purpose computing
- Supports high-bandwidth LPDDR5X memory, high-speed networking, PCIe Gen5 and rich sensor expansion
- Supports NVIDIA robotics and edge AI software ecosystem including JetPack, CUDA, TensorRT, Isaac ROS and Isaac Sim
