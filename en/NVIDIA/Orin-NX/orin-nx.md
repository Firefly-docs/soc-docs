# Orin NX

## Key features

- NVIDIA Jetson Orin NX compact high-performance edge AI module for robotics, drones, machine vision, smart retail and industrial edge computing
- Arm Cortex-A78AE CPU with NVIDIA Ampere GPU and Tensor Cores for energy-efficient AI inference
- Integrated NVDLA deep learning accelerator for multi-stream video analytics and concurrent model inference
- Supports LPDDR5 memory, high-speed camera input, PCIe, USB and display expansion
- Compatible with the Jetson software ecosystem including JetPack, CUDA, TensorRT, DeepStream and Isaac ROS
