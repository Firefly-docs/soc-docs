# AGX Orin

## Key features

- NVIDIA Jetson AGX Orin high-performance edge AI platform for robotics, automation equipment, intelligent transportation and multi-sensor AI systems
- Multi-core Arm Cortex-A78AE CPU with NVIDIA Ampere GPU and Tensor Cores for high-performance heterogeneous computing
- Integrated NVDLA deep learning accelerators and PVA vision accelerator for concurrent models and real-time vision processing
- Supports high-bandwidth LPDDR5 memory, multi-camera input, high-speed storage and rich industrial expansion interfaces
- Supports the NVIDIA edge AI software ecosystem including JetPack, CUDA, TensorRT, DeepStream and Isaac ROS
