# QCS8550

## Key features

- Qualcomm high-performance intelligent IoT processor platform for robotics, industrial terminals, edge AI and smart camera products
- 64-bit Qualcomm Kryo octa-core CPU for both high-performance compute and low-power tasks
- Qualcomm Adreno GPU for graphics rendering, compute and multimedia applications
- Qualcomm Hexagon processor and Qualcomm AI Engine for on-device AI inference and multi-sensor processing
- Qualcomm Spectra ISP for high-resolution cameras, multi-camera input and HDR imaging
- Supports 8K video, high-resolution display, high-speed storage and rich peripheral expansion
