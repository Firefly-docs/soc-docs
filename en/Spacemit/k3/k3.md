# K3

## Key features

- SpacemiT next-generation 64-bit RISC-V processor platform for high-performance computing, edge computing and intelligent terminal products
- Based on SpacemiT self-developed X100 RISC-V CPU core, compatible with the RVA23 software profile
- Supports scalable multi-core cluster configurations for different compute requirements
- Targets the Linux / Bianbu RISC-V software ecosystem, making system and application migration easier
- Final GPU, NPU, multimedia and high-speed interface capabilities depend on the released chip datasheet and board design
