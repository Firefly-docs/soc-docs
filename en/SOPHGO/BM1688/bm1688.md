# BM1688

## Key features

- SOPHGO high-performance processor platform for edge AI, visual analytics, robotics, industrial inspection and intelligent terminals
- Integrated TPU AI accelerator supporting common inference precisions such as INT8 and FP16
- Multimedia processing capability for multi-channel video input, codec workloads and on-device vision inference
- Supports Linux AI development, SOPHON SDK, BModel deployment and conversion flows from mainstream deep learning frameworks
- Suitable for low-power edge devices and compact AI servers; compute, memory and interface details depend on module or board design
