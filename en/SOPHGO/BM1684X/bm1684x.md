# BM1684X

## Key features

- SOPHGO high-performance AI inference processor for edge AI servers, video analytics, smart security and industrial vision
- Provides higher TPU inference performance than BM1684 for larger models and higher-concurrency vision workloads
- Supports multi-precision inference such as INT8, FP16 and FP32 for performance, accuracy and deployment flexibility
- Supports multi-channel video codec, image preprocessing and AI inference pipelines
- Works with SOPHON SDK and BModel deployment flows for migration from mainstream training frameworks to edge inference
