# BM1684

## Key features

- SOPHGO classic TPU processor for AI inference and video analytics in edge servers, smart security and industrial vision
- Integrated TPU accelerator supporting multi-precision inference such as INT8, FP16 and FP32
- Supports multi-channel video codec and image preprocessing for video structuring and vision analytics systems
- Can be integrated through PCIe or board-level interfaces in edge computing devices and servers
- Works with SOPHON SDK for model conversion, compilation, runtime inference and multimedia development
